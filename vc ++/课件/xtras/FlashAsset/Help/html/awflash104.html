Authoring and distributing pieces with the Flash Asset Xtra

The installation program for the Flash Asset Xtra places the following Xtras files in your Authorware Xtras folder:

Windows Macintosh Purpose
Flash Asset Options.x32

 N/A

 This file provides support for Flash movies while authoring movies in Authorware. Don't distribute this file with Authorware pieces. This file isn't licensed for redistribution.
Flash Asset.x32

 Flash Asset PPC

 Distribute this file with any Authorware pieces you create using the Flash Asset Xtra.

In authoring mode if you have only the Flash Asset Xtra—and not the Flash Asset Options Xtra—you can't insert new Flash movies or set the properties of existing movies with the SetSpriteProperty or SetIconProperty functions.

The Windows installation program also provides the latest version of two required Xtras: Mui Dialog.x32 and Actor Control.x32. You need these files, which are part of the standard MuiDialog Xtras package, for authoring only. Don't distribute them with Authorware pieces.

For more information about using Xtras and distributing Authorware pieces made with them, see "Distributing Xtras" in the "Authoring Basics" section of the Authorware 5 Help Pages.
