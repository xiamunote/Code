Distributing pieces with QuickTime VR

To play Authorware pieces that use QuickTime VR, you need:

On Windows: Windows 95, Windows 98, Windows 2000, or Windows NT 4.0 or later
On Macintosh: a computer with a PowerPC processor
QuickTime 3.0 or later (available from www.quicktime.com )

You cannot run Authorware pieces that use QuickTime VR on Windows 3.1 or Macintosh 68K computers.

Packaged Authorware pieces that use QuickTime VR should be distributed with the following Xtra files:

Windows PowerPC Purpose
QuickTime Asset.x32

 QuickTime Asset PPC

 Distribute this file with any packaged pieces you create using the QuickTime Xtra.
MoaFile2.x32

   This file provides support for linking to external QuickTime VR media.

For more information about using Xtras and distributing pieces made with them, see "Using Variables, Functions, Xtras, and More" and "Distributing Xtras" in the Authorware 5 Help Pages.
