frameReady
 
Syntax boolean:=CallSprite(@"IconTitle", #frameReady, framenumber)
 
Type Method
 
Description  This method indicates whether enough of a Flash movie sprite has streamed into memory to render the frame specified in the framenumber parameter. This method returns:

TRUE Enough of the media exists in RAM to render the frame.
FALSE Not enough media exists in RAM to render the frame.

Example  This frame script checks to see if frame 25 of a Flash movie sprite can be rendered.

If CallSprite(@"Flash Icon", #frameReady, 25) then
    FrameReady:=True
else
    FrameReady:=False
end if
