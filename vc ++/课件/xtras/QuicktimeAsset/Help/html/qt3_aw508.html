Using QuickTime Media over the Internet

You can link the QuickTime Media to a file on your disk or to a URL over the internet. If you link to a URL, Authorware begins to download the QuickTime Media when its sprite first appears in the Presentation window.

When using QuickTime Media in pieces that are packaged for the web, keep these points in mind:

The QuickTime Media doesn't appear in the Presentation window until it's fully downloaded. Once it's fully downloaded, it begins to play unless the Paused or Show Controller option is selected in the QuickTime dialog box.
Once QuickTime Media begins to download, the download continues until it's finished, even if the sprite no longer appears in the Presentation window.
Once a QuickTime Media file has finished downloading, its sprite will use the file without downloading it again, as long as the file remains in the file cache.
When scripts or expressions refer to properties or use functions that require information about QuickTime Media that has not finished downloading, the script receives only default information.
