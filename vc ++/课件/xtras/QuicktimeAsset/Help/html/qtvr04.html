Interacting with a QuickTime VR movie

QuickTime VR movies have a set of buttons along the bottom. Unlike those for standard QuickTime movies, these buttons are not the primary means of controlling movies. With QuickTime VR movies, you simply drag across the movie image itself. The buttons on the VR controller supplement this basic action.

Additional features for controlling QuickTime VR movies are available through Authorware scripting. For more information, see the QuickTime VR Scripting Reference.
