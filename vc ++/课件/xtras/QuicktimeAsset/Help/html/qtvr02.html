Introduction to QuickTime VR

QuickTime VR lets you incorporate virtual reality into your Authorware projects. Rotate objects, zoom in or out, look in any direction, and navigate from one scene to another—all without requiring special hardware.

Use Authorware to display and control QuickTime VR movie files. QuickTime VR files are created by applications like Apple Computer's QuickTime VR Authoring Studio.

Because QuickTime VR support is provided in Authorware by the QuickTime Xtra, you can use many of the same controls that apply to standard QuickTime movies. The procedure for importing QuickTime VR movies into Authorware is identical to importing standard QuickTime Media files. See Using QuickTime Media in Authorware for more information.
